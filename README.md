# ORVM
Toy register machine based virtual machine in OCaml.  
  
  
## Build

```zsh
$ git clone https://github.com/alphaKAI/orvm
$ cd orvm
$ jbuilder build
```

## Requirements

* Core
* ppx\_deriving

If you haven't installed them yet, please install them with OPAM.  

```zsh
$ opam install core ppx_deriving
```
